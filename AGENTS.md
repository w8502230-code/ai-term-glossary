# AI 术语普及 — Agent 开发指令

## 1. 项目概述

独立 Web 应用：面向无 IT 背景用户，普及 AI 相关术语。

- **前端**：极简首页（类 Google 首页）+ **输入框右侧语音话筒**（Web Speech API）+ 解释页 + 结束页；移动端优先适配。
- **后端**：AI 代理层，**仅服务端持有百炼 API Key**（在 [百炼控制台](https://bailian.console.aliyun.com/) 管理）；通过 **阿里云百炼** 调用 **通义千问** 生成文案，**不依赖个人 VPN**，**不使用 DeepSeek**。
- **MVP 闭环**：首页输入或点卡片 → 解释页 **专业中 → 专业英 → 通俗（仅中文）** → 继续回首页 / 结束。
- **互动条**：首页与解释页展示 **点赞、转发（符号，参照微信小视频底部）** 与 **`views` 文案 + 浏览数**，产品规则见 [`PRD.md`](PRD.md) §5.6，接口与存储见 [`tech-design.md`](tech-design.md) §5.2。
- **我的收藏**：**解释页**释义成功后 **打勾** 收藏/取消；**首页**「我的收藏」在路由 `/` 上以 **抽屉或全屏层** 打开列表；数据仅 **`localStorage`**（或等价），规则见 [`PRD.md`](PRD.md) §5.7、[`tech-design.md`](tech-design.md) §5.3。

详细需求见 [`PRD.md`](PRD.md)，架构与接口见 [`tech-design.md`](tech-design.md)。

---

## 2. 开发优先级（必须遵守）

1. **先打通闭环**：路由、搜索框（含语音降级）、调用 `/api/define`、展示 **professionalZh / professionalEn / plainZh**、继续/结束。
2. **再保证安全与稳定**：环境变量密钥、输入校验、超时与错误提示、不泄露 Key。
3. **再优化体验**：动效、加载态、文案微调；**互动条（点赞/转发/浏览）** 与后端统计接口对齐；**我的收藏**（打勾、首页列表、localStorage 上限 100）与 PRD §5.7 / `tech-design.md` §5.3 对齐；避免过早抽象。

---

## 3. 开发规范

- 使用 **TypeScript**，保证类型安全。
- 使用**函数式组件 + Hooks**（React）。
- 样式使用 **Tailwind CSS**（与 `tech-design.md` 一致）。
- 组件可复用即可，**避免过度设计**；注释只写「为什么」，不写废话。
- 保持与本文档、`tech-design.md` 的目录与接口约定一致。

---

## 4. 代码风格

- 组件名：`PascalCase`
- 函数/变量：`camelCase`
- 常量：`UPPER_SNAKE_CASE`
- 类型：以 `T` 开头（如 `TDefineResponse`）

---

## 5. 设计要求（UI）

- **风格**：简约、留白多，参考 Google 首页式**居中搜索**布局。
- **配色**：中性浅底 + 单一强调色（具体色值在实现阶段统一即可）；避免花哨背景。
- **圆角/阴影**：适度（如 `rounded-lg`、`shadow` 级别），保持干净。
- **移动端**：触控目标约 **44px** 高；字体与间距在小屏可读。
- **互动条**：与 PRD §5.6 一致——点赞/转发用**符号**（类微信小视频底部），浏览用 **`views` + 数字**；保证 **可点击区域**足够大；点赞/转发需 **aria-label**；`views` 可对读屏补充说明；数字更新时避免整页跳动（可用 `tabular-nums` 等）。
- **我的收藏**：**打勾**表示收藏态；**不得**与点赞使用相同图标；打勾控件需 **aria-label**（如「加入收藏」「取消收藏」）；首页收藏列表为抽屉或全屏层时，注意焦点陷阱与关闭入口（实现阶段与无障碍规范一致即可）。

---

## 6. AI / 后端集成要求（关键）

- **前端禁止**直接调用大模型供应商 SDK；**只请求自有后端**。
- 后端返回结构化字段：`professionalZh`、`professionalEn`、`plainZh`；见 `tech-design.md`。
- 应对模型输出做 **JSON 解析与字段校验**；失败时重试（有限次数）或返回统一错误码/文案。
- **日志中不得打印完整 API Key**；勿将密钥写入仓库。
- 用户输入可为**中文或英文**；输出须含**中文专业段、英文专业段、中文通俗段**（通俗段不要英文版）。释义**仅限 IT / AI / ML / 软件工程语境**，禁止用物理、日常等跨界义项（与 `aiProvider.ts` 系统提示一致）。

---

## 7. 首页：语音与推荐卡片

- **语音**：使用浏览器 **Web Speech API**；不支持或拒绝权限时降级为纯文字；**不得**将用户语音上传自有服务器（当前方案）。
- 术语列表来自**本地静态词池**（JSON），在**每次进入首页**时洗牌后取 **3** 个；不要求调用百炼更新卡片。

---

## 8. 安全与配置

- **百炼**密钥使用环境变量 `BAILIAN_API_KEY`（`.env` 本地、部署平台 Secret）。密钥须在 [百炼控制台](https://bailian.console.aliyun.com/) 创建。
- MVP **可不强制**用户鉴权；若增加中间件，接口形态预留 `Authorization` 扩展即可。
- 对用户输入做**长度上限**与必要净化，降低提示注入与滥用风险。

---

## 9. 测试要求（手动为主）

每轮迭代至少验证：

- 首页加载、输入回车、**话筒识别（在支持的浏览器）**、点击卡片跳转正常。
- 解释页展示专业（中→英）与通俗（中文）三块内容，布局在手机上正常。
- 「继续」回首页后推荐卡片刷新（与词池洗牌逻辑一致）。
- 「结束」进入结束页；可选返回首页。
- 断网或错误接口时有友好提示，无 Key 泄露。
- 首页与解释页：**浏览 / 点赞 / 转发** 显示与交互符合 PRD；解释页仅在释义成功后的浏览上报与 UI 一致；转发在桌面与移动各测一种路径（Share / 复制链接）。
- **我的收藏**：解释页释义成功后打勾可收藏/取消；首页入口可打开列表（**路由仍为 `/`**），列表可跳转解释页、可打勾取消；收藏满 **100** 条后再收藏时淘汰最旧一条；刷新后收藏仍在（同浏览器同站点）；**无后端接口**。
